# Gestor de Citas

Sistema de gestión de citas desarrollado con Node.js, Express y SQLite.
Permite Crear, Leer, y Eliminar citas, además de generar comprobantes en PDF.

## Requisitos
- Node.js instalado

## Instalación y Uso (Cómo hacerlo funcional)
1. Descarga este repositorio (Botón Code > Download ZIP).
2. Abre una terminal en la carpeta del proyecto.
3. Instala las dependencias:
   ```bash
   npm install
